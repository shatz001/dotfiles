How to use this log

Add one <details> block per run (copy the example below). Set the Status paragraph's class to one of pending / running / complete / failed. Paste the launch command verbatim into the <pre><code> block — it wraps, so never truncate it. When the run finishes, flip the status (with finish time + duration) and fill in Results: a one-line summary, and a table if there is tabular output.
